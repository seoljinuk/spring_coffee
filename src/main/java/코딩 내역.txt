테스트 시나리오(get 요청)
    이때, 브라우저는 상품 수정 페이지로 이동해야 하며, 기존에 등록되어 있던 상품의 세부 정보가 모두 입력란에 자동으로 표시되어야 합니다. 예를 들어, 상품명, 가격, 재고 수량, 카테고리, 설명, 이미지 파일명 등의 항목이 기존 데이터와 동일하게 표시되는지를 확인합니다. 그러나 이미지 <input type="file">에는 보안상의 이유로 이전에 업로드한 파일 경로/내용을 기본값으로 넣을 수 없습니다. 이건 브라우저가 파일 입력란에 value를 설정할 수 없도록 막아 놓았기 때문입니다.
